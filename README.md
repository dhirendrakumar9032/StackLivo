# Replit

Read – Eval – Print – Loop

A browser-based React playground and online code editor that allows users to create, manage, and run React applications directly in the browser with live preview support.

---

## ✨ Features

- ⚛️ Create React projects instantly
- 🧠 Live JavaScript execution
- 👀 Real-time preview
- 📝 Monaco code editor integration
- 💾 Save and manage multiple projects
- 🗑️ Delete existing projects
- ⚡ Fast development experience with Vite
- 🎨 Modern responsive UI

---

## 🛠️ Tech Stack

### Frontend
- React.js
- JavaScript
- CSS3
- Monaco Editor
- Vite

### Backend
- Node.js
- Express.js

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/dhirendrakumar9032/Replit.git
```

### Navigate to project

```bash
cd Replit
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```
